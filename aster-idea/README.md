This is IntelliJ IDEA Plugin project for Aster programming language.
## Requirements
* IntelliJ IDEA 2025.1 or later
* Java 25 or later
* Gradle 9 or later
* Aster Language SDK
## Installation
1. Clone this repository.
2. Open the project in IntelliJ IDEA.
3. Build the project using Gradle.
4. Run the plugin in a new IntelliJ IDEA instance using the Gradle task `runIde`.
## Usage
1. Open or create an Aster project in the new IntelliJ IDEA instance.
2. Enjoy the features provided by the Aster plugin, such as syntax highlighting, code completion, and debugging support.
